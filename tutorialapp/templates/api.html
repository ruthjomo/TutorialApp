How to use Award Gallery Api
For better user experience and for developers who would like to use our AwwardMe data, we developed and API who help you out. To use our api we recommend to use it on Postman which is much easier to send the HTTP request i.e GET,POST,PUT,DELETE...

Before you start using our api endpoints you need an Token Autheniticaton only for registered users. On Postman choose the POST http request method on the url place this url www.moringaawards.heroku.com/api-auth-token/ on the body on the postman. Pass you USERNAME and PASSWORD as the key and value. On the request results at the bottom on postman if successful you will get your token.

After token received you can now access user and project data using the following links www.moringaawards.heroku.com/api/users/ and www.moringaawards.heroku.com/api/project/ respectively.
